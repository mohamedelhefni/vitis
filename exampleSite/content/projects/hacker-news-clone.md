## Overview

A clone of Hacker News that uses the official API with React.js and Tailwind CSS.

## Features

- Display news and stories
- Search box
- Filter for comments and stories
- Clean and responsive design

## Technologies

- React.js with Hooks
- Tailwind CSS for styling
- Hacker News API
