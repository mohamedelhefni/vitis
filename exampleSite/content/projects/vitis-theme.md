Vitis is a clean, minimal Hugo theme designed for knowledge management and digital gardening. Built with a focus on content connections and visual knowledge representation.

## Key Features

- Interactive knowledge graph visualization
- Wiki-style bidirectional links
- Series support for tutorial content
- Dark mode with no FOUC
- Full-text search
- Responsive design
- Fast and lightweight

## Tech Stack

- Hugo (Static Site Generator)
- D3.js (Knowledge Graph)
- Tailwind CSS
- Vanilla JavaScript
