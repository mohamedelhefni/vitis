## Overview

A simple diary app built with Vanilla JavaScript and Tailwind CSS.

## Features

- Add new note
- Automatically edit today's note
- Review and edit old notes
- Add mood (happy, normal, sad)
- Delete notes
- Download all notes as PDF file
